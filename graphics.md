# 图形系统设计
## 屏幕规范
屏幕设计为标准 5128256:
```
        │
        │
        │
        │0
────────┼─────────────────────────────────────────────┬─────► X
        │                                             │ 512
        │                                             │
        │                                             │
        │                                             │
        │                                             │
        │                                             │
        │                hello world                  │
        │                                             │
        │                                             │
        │                                             │
        │                                             │
        │                                             │
        │                                             │
 256    ├─────────────────────────────────────────────┘
        │
        │
        │
        │
        ▼ Y
```
## 指令
基础指令为画点指令，可以利用其扩展成画线，图片等等：
```sh
    DP XYC
```
表示在 (X，Y) 处画一个点，其颜色为 C。
其中指令码为1字节,参数：
- X：X坐标,1 字节

- Y：Y坐标,1 字节

- C：颜色，1字节，目前规范了 32种颜色，见下表：
